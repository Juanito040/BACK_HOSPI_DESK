# CLI DE EJEMPLO

El proyecto presenta una CLI(command line interface), la cual ha sido construida con las siguientes herramientas:

- TypeScript
- Node.js
- TS-Node (En modo desarrollo)
- Colors
- Inquire

Y siguiendo los conceptos de :

- POO (programacion orientada a objetos)
- Clean Archictecture.
- Principios SOLID.
